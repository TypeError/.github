### About

TypeError is a collection of cybersecurity projects that explore how we build, break, and better secure software.  
Its repositories span **application security**, **vulnerability research**, and **data science**, turning complex data into insight that drives real decisions.

Each project follows the same principles that define good security engineering: clarity, reliability, and scalability.  
From red-team utilities to defensive analytics and automation frameworks, TypeError projects aim to make security easier to explain, test, and trust.
